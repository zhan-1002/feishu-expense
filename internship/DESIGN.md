# 实习工时模块设计文档

## 1. 需求概述

### 1.1 背景
实习生每周填写出勤记录，每月汇总后提交财务审批，用于计算实习补贴。

### 1.2 核心规则

| 项目 | 规则 |
|------|------|
| 大小周判断 | 以周六所在月份判断，该月第二周周六为大周（上班） |
| 出勤粒度 | 全天/上午/下午/无 |
| 工时计算 | 跨月周按实际日期归属月份拆分计算 |

### 1.3 角色

| 角色 | 职责 |
|------|------|
| 实习生 | 填写周出勤记录 |
| 财务 | 审批月度工时统计 |

## 4. 大小周计算规则

### 4.1 判断逻辑

```python
def get_week_of_month(saturday_date: date) -> int:
    """获取周六是该月第几周
    
    规则：以周六所在月份判断，该月第二周的周六为大周
    """
    # 获取周六所在月份
    month = saturday_date.month
    year = saturday_date.year
    
    # 找到该月第一个周六
    first_day = date(year, month, 1)
    days_until_saturday = (5 - first_day.weekday()) % 7
    first_saturday = first_day + timedelta(days=days_until_saturday)
    
    # 计算当前周六是第几个周六
    weeks_diff = (saturday_date - first_saturday).days // 7
    return weeks_diff + 1  # 第N周（从1开始）

def is_big_week(saturday_date: date) -> bool:
    """判断是否为大周（上班）
    
    大周：该月第二周的周六
    小周：其他周六
    """
    week_num = get_week_of_month(saturday_date)
    return week_num == 2
```

### 4.2 示例

以 2024年5月 为例：

| 日期 | 星期 | 周数 | 大小周 |
|------|------|------|--------|
| 5/4 | 周六 | 第1周 | 小周 |
| 5/11 | 周六 | 第2周 | **大周** |
| 5/18 | 周六 | 第3周 | 小周 |
| 5/25 | 周六 | 第4周 | 小周 |

## 5. 跨月周工时计算

### 5.1 规则

周记录可能横跨两个月（如周三到下周一），工时按日期归属月份拆分：

```
示例：2024年4月29日(周一) - 5月5日(周日)

归属4月的天数：4/29, 4/30 → 2天
归属5月的天数：5/1, 5/2, 5/3, 5/4, 5/5 → 5天

工时计算时：
- 4月统计：只计算 4/29, 4/30 的出勤
- 5月统计：只计算 5/1-5/5 的出勤
```

### 5.2 实现逻辑

```python
def calculate_monthly_attendance(records: List[Dict], year: int, month: int) -> float:
    """计算指定月份的出勤天数
    
    Args:
        records: 周出勤记录列表
        year: 目标年份
        month: 目标月份
    
    Returns:
        出勤天数（半天=0.5）
    """
    total_days = 0.0
    
    for record in records:
        week_start = record['week_start_date']
        week_end = record['week_end_date']
        
        # 遍历每一天
        current = week_start
        while current <= week_end:
            if current.year == year and current.month == month:
                # 获取该天的出勤状态
                day_name = ['周一', '周二', '周三', '周四', '周五', '周六', '周日'][current.weekday()]
                status = record.get(day_name, '无')
                
                # 计算天数
                if status == '全天':
                    total_days += 1.0
                elif status in ('上午', '下午'):
                    total_days += 0.5
            
            current += timedelta(days=1)
    
    return total_days
```

## 6. 日历集成

### 6.1 节假日判断

使用飞书日历 API 获取：

```python
def is_workday(date: date) -> bool:
    """判断是否为工作日
    
    1. 周一至周五默认为工作日
    2. 查询飞书日历是否有节假日标记
    3. 查询飞书日历是否有调休安排（周末上班）
    """
    weekday = date.weekday()
    
    # 周末默认不上班
    if weekday >= 5:  # 周六、周日
        # 检查是否有调休安排
        if has_adjustment_workday(date):
            return True
        return False
    
    # 工作日默认上班
    # 检查是否为节假日
    if is_holiday(date):
        return False
    
    return True
```

### 8.2 触发判断

```python
def should_create_weekly_records(now: datetime) -> bool:
    """判断是否应创建周记录
    
    条件：本周最后工作日 14:00
    """
    # 找到本周最后工作日
    weekday = now.weekday()
    last_workday_offset = 4 - weekday  # 周五=4
    last_workday = now.date() + timedelta(days=last_workday_offset)
    
    # 判断是否为14:00前后5分钟
    target_time = datetime.combine(last_workday, time(14, 0))
    return abs((now - target_time).total_seconds()) < 300
```

## 9. 错误处理

### 9.1 异常场景

| 场景 | 处理方式 |
|------|----------|
